# Quickstart Guide

Get your Coffetto website up and running in just a few minutes. This guide will walk you through downloading, opening, and customizing your coffee shop website.

## Prerequisites

All you need is:
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- A text editor (VS Code, Sublime Text, or any code editor)
- Basic knowledge of HTML and CSS (helpful but not required)

<Steps>
  <Step title="Download Coffetto">
    Clone or download the Coffetto repository to your local machine.

    ```bash
    git clone <repository-url>
    cd coffetto
    ```

    Or simply download the ZIP file and extract it to your preferred location.
  </Step>

  <Step title="Open in Browser">
    The simplest way to view Coffetto is to open the `index.html` file directly in your browser.

    **Option 1: Direct Open**
    - Navigate to the project folder
    - Double-click `index.html`
    - The website opens in your default browser

    **Option 2: Local Server (Recommended)**
    
    For a better development experience, use a local server:

    <CodeGroup>
    ```bash Python
    # Using Python 3
    python -m http.server 8000
    ```

    ```bash Node.js
    # Using npx (comes with Node.js)
    npx serve
    ```

    ```bash PHP
    # Using PHP
    php -S localhost:8000
    ```
    </CodeGroup>

    Then open `http://localhost:8000` in your browser.

    <Tip>
    Using a local server prevents CORS issues and provides a more production-like environment.
    </Tip>
  </Step>

  <Step title="Customize Content">
    Now it's time to make Coffetto your own! Here are the key areas to customize:

    ### Update Brand Name
    
    Replace "Coffetto" with your coffee shop name in `index.html`:

    ```html index.html
    <a href="#" class="nav__logo">
        <img src="assets/img/logo.png" alt="logo">
        Coffetto  <!-- Change this to your brand name -->
    </a>
    ```

    ### Edit Home Section
    
    Update the hero section with your message:

    ```html index.html
    <h3 class="home__subtitle">
        CALIDAD EXCEPCIONAL  <!-- Your tagline -->
    </h3>

    <h1 class="home__title">
        Es hora de un <br>
        buen café  <!-- Your main headline -->
        <img src="assets/img/home-coffee-title.png" alt="home image">
    </h1>

    <p class="home__description">
        Cada grano de café selecto refleja nuestro 
        compromiso con los cafetaleros peruanos...
        <!-- Your brand description -->
    </p>
    ```

    ### Update Products
    
    Modify the product cards to showcase your offerings:

    ```html index.html
    <article class="products__card">
        <img src="assets/img/product-coffee-1.png" alt="products image" class="products__img">
        <h3 class="products__name">Café clásico</h3>  <!-- Product name -->
        <span class="products__price">S/17.90</span>   <!-- Product price -->
    </article>
    ```

    <Note>
    The template includes three product cards. You can duplicate or remove cards by copying/deleting the `<article class="products__card">` elements.
    </Note>

    ### Customize Navigation
    
    Update navigation labels to match your content:

    ```html index.html
    <li class="nav__item">
        <a href="#home" class="nav__link active-link">
            <i class="ri-home-5-fill"></i>
            <span>Inicio</span>  <!-- Change navigation text -->
        </a>
    </li>
    ```
  </Step>

  <Step title="Modify Colors & Styling">
    Coffetto uses CSS custom properties for easy theming. All color variables are defined in `assets/css/styles.css`.

    ### Color Variables
    
    Update the color scheme by modifying these variables in the `:root` selector:

    ```css assets/css/styles.css
    :root {
      /* Primary colors */
      --first-color: hsl(12, 32%, 22%);        /* Main dark brown */
      --first-color-alt: hsl(12, 42%, 12%);    /* Darker variant */
      --first-color-light: hsl(12, 24%, 32%);  /* Lighter variant */
      --second-color: hsl(39, 62%, 65%);       /* Golden accent */
      
      /* Text colors */
      --title-color: hsl(12, 12%, 16%);
      --text-color: hsl(12, 12%, 32%);
      --text-color-light: hsl(12, 8%, 48%);
      --white-color: #fff;
    }
    ```

    ### Typography
    
    Adjust font sizes using these variables:

    ```css assets/css/styles.css
    :root {
      --body-font: "Poppins", sans-serif;
      --second-font: "Pacifico", cursive;
      --biggest-font-size: 2.75rem;
      --big-font-size: 1.75rem;
      --h1-font-size: 1.5rem;
      --h2-font-size: 1.25rem;
      --h3-font-size: 1rem;
      --normal-font-size: .938rem;
    }
    ```

    <Tip>
    Changing the CSS variables in the `:root` selector will update colors and fonts throughout the entire website automatically.
    </Tip>

    ### Border Radius
    
    Customize the curved section borders:

    ```css assets/css/styles.css
    :root {
      --border-section: 56px 56px 0 0;  /* Adjust for more/less curve */
    }
    ```
  </Step>

  <Step title="Add Your Images">
    Replace the default images with your own coffee shop photos:

    1. **Logo** - Replace `assets/img/logo.png` (28px × 28px recommended)
    2. **Favicon** - Replace `assets/img/favicon.png` (16px × 16px or 32px × 32px)
    3. **Hero Image** - Replace `assets/img/home-coffee.png` (550px wide recommended)
    4. **Product Images** - Replace `assets/img/product-coffee-1.png`, `product-coffee-2.png`, `product-coffee-3.png`
    5. **About Image** - Replace `assets/img/about-coffee.png` (550px wide recommended)
    6. **Background** - Replace `assets/img/coffee-beans-bg.png` for section backgrounds

    <Warning>
    Keep the same file names when replacing images, or update the `src` attributes in `index.html` to match your new filenames.
    </Warning>

    ### Image Format Recommendations
    
    - Use **PNG** for images with transparency (logo, product images)
    - Use **JPG** for photographs (about section, testimonial)
    - Use **SVG** for icons and decorative elements (like `steps-curve-line.svg`)
  </Step>
</Steps>

## Testing Your Changes

After customizing, test your website:

1. **Responsive Design** - Use browser DevTools to test different screen sizes
2. **Scroll Effects** - Scroll through the page to verify animations work
3. **Navigation** - Click navigation links to ensure smooth scrolling
4. **Scroll-to-Top** - Scroll down and verify the scroll-up button appears

<Tip>
Press `F12` or `Cmd+Option+I` (Mac) to open browser DevTools and test responsive design at different breakpoints: 300px, 576px, 1023px, and 1248px.
</Tip>

## Need Help?

If you encounter any issues during setup:

- Check that all files are in the correct directory structure
- Verify image paths match your actual filenames
- Make sure you're using a modern browser with JavaScript enabled
- Try using a local server instead of opening the HTML file directly
