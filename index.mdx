    </p>
  </div>

  <Steps>
    <Step title="Clone the repository">
      Download the Coffetto website template to your local machine:
      
      ```bash
      git clone https://github.com/yoelfh/web-cafe.git
      cd web-cafe
      ```
    </Step>
    
    <Step title="Open in your browser">
      Launch the website by opening the `index.html` file in any modern web browser. No build process or dependencies required!
      
      ```bash
      # Using Python's built-in server (optional)
      python -m http.server 8000
      
      # Or using Node.js http-server (optional)
      npx http-server
      ```
      
      <Note>
        The website is built with vanilla HTML, CSS, and JavaScript—no framework required. Just open `index.html` directly or serve it with any static file server.
      </Note>
    </Step>
    
    <Step title="Customize the content">
      Edit the content in `index.html` to match your coffee shop's branding:
      
      - Update the brand name (search for "Coffetto")
      - Replace product names and prices
      - Modify section text in Spanish or translate to your language
      - Add your own images in the `assets/img/` directory
      
      All changes appear instantly when you refresh the browser.
    </Step>
  </Steps>
