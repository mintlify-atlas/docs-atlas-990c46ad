### Product Cards

Each product has this structure (example: lines 114-118):

```html
<article class="products__card">
  <img src="assets/img/product-coffee-1.png" alt="products image" class="products__img">
  <h3 class="products__name">Classic Coffee</h3>
  <span class="products__price">S/17.90</span>  <!-- S/ is Peruvian Sol -->
</article>
```

### Customizing Products

<Tabs>
  <Tab title="Edit Existing">
    Replace the content in each product card:
    ```html
    <article class="products__card">
      <img src="assets/img/your-product.png" alt="product name" class="products__img">
      <h3 class="products__name">Your Product Name</h3>
      <span class="products__price">$12.99</span>  <!-- Use your currency -->
    </article>
    ```
  </Tab>
  
  <Tab title="Add More Products">
    Copy an entire `<article>` block and paste it within the `.products__content` div:
    ```html
    <article class="products__card">
      <img src="assets/img/product-coffee-4.png" alt="new product" class="products__img">
      <h3 class="products__name">New Product</h3>
      <span class="products__price">$15.99</span>
    </article>
    ```
  </Tab>
  
  <Tab title="Remove Products">
    Delete the entire `<article>` block for any product you don't want
  </Tab>
</Tabs>

**Product image dimensions**: 120x120px (mobile), scales up on desktop

<Tip>
  If you add more than 3 products, consider adjusting the grid layout. See [Layout Configuration](/customization/layout-configuration#products-grid).
</Tip>

## Image Replacement Guide

### Image Specifications

| Location | Element | Recommended Size | Format |
|----------|---------|------------------|--------|
| Logo | Header/Footer | 28-32px square | PNG (transparent) |
| Favicon | Browser tab | 16x16 or 32x32px | PNG/ICO |
| Hero image | Home section | 320-550px | PNG (transparent) |
| Hero title image | Home title | 80-120px | PNG (transparent) |
| Product images | Products | 120px square | PNG (transparent) |
| About image | About section | 320-550px | PNG/JPG |
| Step images | Steps | 120-210px | PNG (transparent) |
| Testimonial | Testimonial | 320-550px | PNG/JPG |
| Background beans | Footer/Steps | Variable | PNG (transparent) |

### Optimization Tips

<Steps>
  <Step title="Compress images">
    Use tools like TinyPNG, ImageOptim, or Squoosh to reduce file size without quality loss
  </Step>
  
  <Step title="Use appropriate formats">
    - **PNG**: For images with transparency (logos, product images)
    - **JPG**: For photographs without transparency
    - **WebP**: Modern format for better compression (with JPG/PNG fallback)
  </Step>
  
  <Step title="Provide retina versions">
    For crisp display on high-DPI screens, export @2x versions at double dimensions
  </Step>
  
  <Step title="Update alt text">
    Always include descriptive alt text for accessibility
  </Step>
</Steps>

## SEO Metadata

Add meta tags in the `<head>` section for better SEO:

```html
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta name="description" content="Your coffee shop description for search engines">
  <meta name="keywords" content="coffee, specialty coffee, artisan roasting">
  <meta name="author" content="Your Brand Name">
  
  <!-- Open Graph for social sharing -->
  <meta property="og:title" content="Your Coffee Shop Name">
  <meta property="og:description" content="Crafting exceptional coffee since...">
  <meta property="og:image" content="https://yoursite.com/og-image.jpg">
  <meta property="og:url" content="https://yoursite.com">
  
  <!-- Existing content -->
  <link rel="shortcut icon" href="assets/img/favicon.png" type="image/x-icon">
  <!-- ... -->
</head>
```

## Content Best Practices

<Tip>
  **Writing effective content:**
  
  1. **Be concise** - Mobile users scan quickly
  2. **Use active voice** - "We roast" vs "Coffee is roasted"
  3. **Show personality** - Let your brand voice shine
  4. **Include keywords** - Naturally incorporate SEO terms
  5. **Add calls-to-action** - Guide users to next steps
  6. **Update regularly** - Keep testimonials and products current
  7. **Proofread** - Check spelling and grammar
  8. **Test readability** - Use tools like Hemingway Editor
</Tip>

<Warning>
  **Common mistakes to avoid:**
  
  - Using Lorem Ipsum placeholder text in production
  - Breaking HTML structure when editing
  - Forgetting to update alt text on images
  - Leaving default social media links
  - Using images that are too large (slow loading)
  - Inconsistent tone between sections
</Warning>
